# E-Katalog Projesi

## Kısa Açıklama

Bu proje, kullanıcıların ürünleri kategorize edilmiş bir şekilde inceleyebildiği ve detaylarına ulaşabildiği kapsamlı bir E-Katalog uygulamasıdır. 

**Öne Çıkan Özellikler:**
* * * Sepet Yönetimi: * * Kullanıcılar ürünleri sepetlerine ekleyebilir. Sepet verileri yerel depolama (Local Storage) üzerinde saklandığı için uygulama kapatılıp açılsa dahi veriler korunur.
* * * Ödeme ve Kart Kaydı: * * Ödeme aşamasında yeni kredi/banka kartı bilgileri sisteme tanımlanabilir ve bu bilgiler sonraki işlemler için güvenli bir şekilde yerel veritabanında saklanabilir.
* * * Kategori Bazlı Listeleme: * * Ürünler türlerine göre filtrelenerek kullanıcı dostu bir arayüzle sunulur.

## Kullanılan Flutter Sürümü
- Flutter: 3.41.3
- Dart: 3.11.1

## Çalıştırma Adımları
1. Bu repository'yi bilgisayarınıza klonlayın:
   ```bash
   git clone https://github.com/Talhaozt/E-Katalog.git

2. Bu proje dizinine gidiniz:
  cd E-Katalog

3. Gerekli paketleri indiriniz:
   flutter pub get

4. Uygulamayı Çalıştırın:
(Bağlı bir emülatör veya gerçek cihaz olduğundan emin olun)
  flutter run
